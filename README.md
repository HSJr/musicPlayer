# HTML5 音樂撥放器-練習 musicPlayer-pratise
[音樂撥放器線上看](https://hsjr.github.io/musicPlayer/) 版權因素故無放置音樂

# 主要使用技術
  * HTML
  * CSS
  * JavaScript
# 主要功能
  * 播放、暫停、停止
  * 前進、倒退
  * 上一首、下一首
  * 隨機撥放、循環撥放、單曲循環
  * 音量調節
  * 靜音
  * 顯示目前播放狀態
  * 歌曲清單
